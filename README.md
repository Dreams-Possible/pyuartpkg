# pyuartpkg
一个基于 pyserial 的全平台串口数据包收发库

## 这是什么，有什么用
* 这是一个基于 pyserial 的二次封装串口库
* 它可以将数据打包并通过串口发送
* 它可以通过串口接收数据并解包

## 它能干什么，有什么好处
* 它将数据封装成数据包，而不是直接传输，这将带来许多益处
* 它自动管理数据包的解打包，提供统一的格式和表现，并且无需用户关注实际的底层实现
* 支持任意格式的数据，例如最基础的帧化二进制格式
* 支持理论无上限的数据包大小，实际上取决于你的电脑能为python分配的最大内存

## 它有什么不同
* 不同于 pyserial，它提供更上层、更稳定、更现代化的通信方式
* 不同于 rosserial，它不依赖于任何第三方环境，也不强制要求任何数据格式

## 它如何做的
* 将数据封包，并使用 HDLC/SLIP 风格的字节转义，这将确保数据包具有唯一可还原性
* 引入了动态缓冲区，这样就可以灵活的处理被随机分片的数据包，这将显著提高数据在实时操作系统（时间不敏感）上的传输稳定性
